# Palamau Tiger Reserve — Official Journey Planner

> Government of Jharkhand | Department of Forest, Environment & Climate Change

Official eco-tourism web app for Palamau Tiger Reserve, Jharkhand. Built for Deputy Director Prajeshkant Jena IFS.

## Stack
React 18 · Vite · Tailwind CSS · Leaflet (OpenStreetMap) · PWA

## Run locally
```bash
npm install
npm run dev
```
